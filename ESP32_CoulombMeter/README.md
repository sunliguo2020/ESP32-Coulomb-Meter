# ESP32 库仑计 - 远程开关 - 功率计

## 项目规格参数

| 参数 | 值 |
|------|-----|
| 输入电压 | 12 ~ 96.0V |
| 输出电压 | 12 ~ 96.0V |
| 输出电流 | 0 ~ 80.0A (取决于采样电阻) |
| 远程开关 | 5路 |
| BMS串口 | 1路 (接电池保护板) |
| 温度监控 | 1路 (NTC) |
| 采样电阻配置 | 1mΩ=80A, 2mΩ=40A, 4mΩ=20A |

## 硬件设计

基于网络表 `Netlist_ESP32_库仑计_2026-06-08.enet`:

- **主控**: ESP32 (U2)
- **屏幕**: TFT 1.8寸 (CN8) SPI接口 (7针, 软件SPI)
- **电流检测**: INA226 (U70) MSOP-10封装, I2C接口
- **采样电阻**: 2个2512封装采样电阻并联 (默认2mΩ, 最大77.5A)
- **电源**: MP9486A (U3) 高压降压 + AMS1117-3.3 (U5) 稳压
- **NTC温度检测** (GPIO39)
- **BMS串口** (UART1: RX=GPIO16, TX=GPIO17) 接电池保护板
- **5路继电器输出** (GPIO14/27/4/33/32)

## 引脚连接 (基于网络表 Netlist_ESP32_库仑计_2026-06-08.enet)

### I2C - INA226
| 信号 | ESP32引脚 | 网络表连接 |
|------|-----------|-----------|
| SDA | GPIO21 | U2.33 → SDA → U70.4 |
| SCL | GPIO22 | U2.36 → SCL → U70.5 |

### TFT 屏幕 (1.8寸 SPI) - CN8
| 信号 | ESP32引脚 | CN8引脚 | 网络表连接 |
|------|-----------|---------|-----------|
| GND | - | CN8.1 | GND |
| VCC | - | CN8.2 | 3.3V |
| SCK | GPIO15 | CN8.3 | U2.23 |
| SDA(MOSI) | GPIO13 | CN8.4 | U2.16 |
| RES | GPIO25 | CN8.5 | U2.10 |
| DC | GPIO26 | CN8.6 | U2.11 |
| BLK | GPIO12 | CN8.7 | U2.14 |

> 注: CS引脚接地常开 (代码中 `#define TFT_CS -1`)，适用于7针SPI模块。

### BMS串口 (接电池保护板) - H1
| 信号 | H1引脚 | ESP32引脚 | 网络表连接 |
|------|--------|-----------|-----------|
| 3.3V | H1.1 | - | 3.3V输出 |

## 采样电阻配置

PCB上使用2个2512封装采样电阻并联，默认总阻值约2mΩ。
根据实际焊接的采样电阻值，修改代码中的宏定义:

```cpp
// 1mΩ → 最大80A (默认)
#define SHUNT_RESISTOR  0.001
#define MAX_CURRENT     80.0

// 2mΩ → 最大40A
// #define SHUNT_RESISTOR  0.002
// #define MAX_CURRENT     40.0

// 4mΩ → 最大20A
// #define SHUNT_RESISTOR  0.004
// #define MAX_CURRENT     20.0
```

## 点灯科技(Blinker)配置

### 一键配网模式（推荐）

代码已配置为一键配网模式，无需在代码中写死WiFi密码：

1. 手机安装 **点灯科技Blinker** App
2. 打开App → 点击右上角 **+** → **独立设备** → **网络接入** → 选择 **ESP32**
3. 复制生成的 **Auth Key**，替换代码中的 `auth[]`：
   ```cpp
   char auth[] = "这里填你的密钥";
   ```
4. 编译上传固件到ESP32
5. **首次配网**：ESP32启动后会发出名为 `Blinker_xxx` 的热点
6. 手机连接该热点 → 打开Blinker App → 自动弹出配网页面
7. 输入你家WiFi的名称和密码 → 配网成功
8. 以后ESP32会自动连接WiFi，无需重复配网

> **更换WiFi方法**：使用串口命令 `AT+WIFI.connect WiFi名称 WiFi密码`

## 串口AT指令

通过USB串口发送以下命令：

| 命令 | 功能 |
|------|------|
| `AT+HELP` | 显示帮助 |
| `AT+RST` | 重启设备 |
| `AT+REG 激活码` | 产品激活 |
| `AT+WIFI.connect SSID 密码` | 连接WiFi |
| `AT+INAV 实际电压` | 设置电压校准 |
| `AT+INAC 实际电流` | 设置电流校准 |
| `AT+RESTORE` | 恢复出厂设置 |

## 功能特性

- ✅ 实时电压/电流/功率测量 (INA226, 精度0.1%)
- ✅ 电池电量百分比显示 (支持铁锂/三元/铅酸)
- ✅ 累计AH/KWh计量
- ✅ 5路远程开关控制 (点灯科技APP)
- ✅ NTC温度监控
- ✅ BMS串口读取 (UART1, 接电池保护板)
- ✅ OTA固件升级
- ✅ 数据掉电保存 (Preferences)
- ✅ 本地按键操作 (MENU/LEFT/RIGHT)
- ✅ 1.8寸TFT彩屏显示 (160x128)
- ✅ 功率曲线图表
- ✅ 电压/电流校准 (串口AT指令)
