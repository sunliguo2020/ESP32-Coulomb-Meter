# ESP32 库仑计 - 远程开关 - 功率计

## 项目规格参数

| 参数 | 值 |
|------|-----|
| 输入电压 | 12 ~ 96.0V |
| 输出电压 | 12 ~ 96.0V |
| 输出电流 | 0 ~ 80.0A (取决于采样电阻) |
| 远程开关 | 5路 |
| 串口读取 | 1路 |
| 温度监控 | 1路 |
| 采样电阻配置 | 1mΩ=80A, 2mΩ=40A, 4mΩ=20A |

## 硬件设计

基于网络表 `Netlist_ESP32_库仑计_2026-05-30.tel`:

- **主控**: ESP32 (U2)
- **屏幕**: TFT 1.8寸 (CN4) SPI接口
- **电流检测**: INA226 (U70) MSOP-10封装, I2C接口
- **采样电阻**: 默认1mΩ (最大80A), 可改为2mΩ(40A)或4mΩ(20A)
- **运放**: U71-U74 电流信号调理
- **电源**: MP9486A (U3) 降压 + AMS1117-3.3 (U5) 稳压
- **NTC温度检测** (H16)
- **风扇PWM控制** (GPIO16)
- **5路继电器输出**

## 点灯科技(Blinker)配置

### 一键配网模式（推荐）

代码已配置为一键配网模式，无需在代码中写死WiFi密码：

1. 手机安装 **点灯科技Blinker** App
2. 打开App → 点击右上角 **+** → **独立设备** → **网络接入** → 选择 **ESP32**
3. 复制生成的 **Auth Key**，替换代码中的 `auth[]`：
   ```cpp
   char auth[] = "这里填你的密钥";
   ```
4. 编译上传固件到ESP32
5. **首次配网**：ESP32启动后会发出名为 `Blinker_xxx` 的热点
6. 手机连接该热点 → 打开Blinker App → 自动弹出配网页面
7. 输入你家WiFi的名称和密码 → 配网成功
8. 以后ESP32会自动连接WiFi，无需重复配网

> **更换WiFi方法**：长按 **EN按键（GPIO34）** 3秒，屏幕显示"WiFi配网模式"，ESP32会清除已保存的WiFi信息并重新发出热点，然后用手机连接该热点，在Blinker App中重新配置WiFi即可。
