## 四、SPI 通信时序

- 接口: 4线 SPI (CS, SCL, SDA, DC)
- 数据传输: MSB first (高位先传)
- 8位数据长度
- 写数据时: DC = 高电平
- 写命令时: DC = 低电平

### SPI 写字节时序 (软件模拟)

```
CS\__/---------------------------\__
       ___     ___     ___
SCL __|   |___|   |___|   |______
         \     \     \
SDA  -----D7----D6----D0---------
```

### 命令/数据控制

```
DC = 0: 写入的是命令 (寄存器)
DC = 1: 写入的是数据
```

## 五、ST7789 初始化序列

以下为 1.54寸 (240×240) ST7789 的标准初始化代码:

### 5.1 完整初始化流程

```c
// ==== ST7789 初始化函数 ====
void TFTLCD_Init(void)
{
    // 1. GPIO 初始化
    TFTLCD_GPIO_Init();

    // 2. 硬件复位
    TFTLCD_RST_RESET;   // 复位低电平
    delay_ms(100);
    TFTLCD_RST_SET;     // 复位高电平
    delay_ms(100);

    // 3. 开启背光
    TFTLCD_BLK_SET;
    delay_ms(100);

    // ==== 以下是 ST7789 初始化命令序列 ====

    // ---- 3.1 存储器访问控制 (MADCTL) ----
    // 寄存器 0x36，控制显示方向
    // USE_HORIZONTAL=0: 0x00 (正常)
    // USE_HORIZONTAL=1: 0x60 (X镜像)
    // USE_HORIZONTAL=2: 0xA0 (Y镜像)
    // USE_HORIZONTAL=3: 0xC0 (XY镜像)
    WriteCmd(0x36);
    WriteData(0x00);

    // ---- 3.2 像素格式 ----
    // 0x3A: 设置颜色深度
    // 0x05 = 16位色 (RGB565)
    WriteCmd(0x3A);
    WriteData(0x05);

    // ---- 3.3  porch 设置 ----
    WriteCmd(0xB2);
    WriteData(0x0C);
    WriteData(0x0C);
    WriteData(0x00);
    WriteData(0x33);
    WriteData(0x33);

    // ---- 3.4 栅极控制 ----
    WriteCmd(0xB7);
    WriteData(0x35);

    // ---- 3.5 VCOM 设置 ----
    WriteCmd(0xBB);
    WriteData(0x19);

    // ---- 3.6 电源控制 ----
    WriteCmd(0xC0);
    WriteData(0x2C);

    WriteCmd(0xC2);
    WriteData(0x01);

    WriteCmd(0xC3);
    WriteData(0x12);

    WriteCmd(0xC4);
    WriteData(0x20);

    WriteCmd(0xC6);
    WriteData(0x0F);

    // ---- 3.7 电源控制 ----
    WriteCmd(0xD0);
    WriteData(0xA4);
    WriteData(0xA1);

    // ---- 3.8 Gamma 校正 (正极) ----
    WriteCmd(0xE0);
    WriteData(0xD0);
    WriteData(0x04);
    WriteData(0x0D);
    WriteData(0x11);
    WriteData(0x13);
    WriteData(0x2B);
    WriteData(0x3F);
    WriteData(0x54);
    WriteData(0x4C);
    WriteData(0x18);
    WriteData(0x0D);
    WriteData(0x0B);
    WriteData(0x1F);
    WriteData(0x23);

    // ---- 3.9 Gamma 校正 (负极) ----
    WriteCmd(0xE1);
    WriteData(0xD0);
    WriteData(0x04);
    WriteData(0x0C);
    WriteData(0x11);
    WriteData(0x13);
    WriteData(0x2C);
    WriteData(0x3F);
    WriteData(0x44);
    WriteData(0x51);
    WriteData(0x2F);
    WriteData(0x1F);
    WriteData(0x1F);
    WriteData(0x20);
    WriteData(0x23);

    // ---- 3.10 进入睡眠模式 ----
    WriteCmd(0x21);  // 显示反转开启

    // ---- 3.11 退出睡眠 ----
    WriteCmd(0x11);  // Sleep out
    delay_ms(120);

    // ---- 3.12 开启显示 ----
    WriteCmd(0x29);  // Display on

    // ---- 3.13 设置初始显示区域 (240×240) ----
    // 设置列地址 (X)
    WriteCmd(0x2A);
    WriteData(0x00);  // 起始列高位
    WriteData(0x00);  // 起始列低位
    WriteData(0x00);  // 结束列高位
    WriteData(0xEF);  // 结束列低位 (240-1=0xEF)

    // 设置行地址 (Y)
    WriteCmd(0x2B);
    WriteData(0x00);  // 起始行高位
    WriteData(0x00);  // 起始行低位
    WriteData(0x01);  // 结束行高位
    WriteData(0x3F);  // 结束行低位 (320-1=0x13F? 实际此屏240-1=0xEF)

    // 开始写显存
    WriteCmd(0x2C);  // Memory write
}
```

## 六、常用寄存器说明

### 6.1 寄存器 0x36 - MADCTL (存储访问控制)

控制 GRAM 指针增长方向，从而控制显示方向。

| 位 | 描述 |
|----|------|
| Bit D7 (MY) | 0: 页显示从顶部往底部<br>1: 页显示从底部往顶部 |
| Bit D6 (MX) | 0: 纵向显示从左往右<br>1: 纵向显示从右往左 |
| Bit D5 (MV) | 0: 正常显示<br>1: 翻转显示 (行列交换) |
| Bit D4 (ML) | 0: 刷新从左到右<br>1: 刷新从右到左 |
| Bit D3 (BGR) | 0: RGB 顺序<br>1: BGR 顺序 |
| Bit D2 (MH) | 0: 水平刷新从左到右<br>1: 水平刷新从右到左 |

**常用方向设置值:**

| 方向 | 寄存器值 | 说明 |
|------|---------|------|
| 正常 (0°) | 0x00 | 标准方向 |
| X镜像 | 0x60 | 水平翻转 |
| Y镜像 | 0xA0 | 垂直翻转 |
| XY镜像 | 0xC0 | 180°翻转 |

### 6.2 常用寄存器列表

| 寄存器 | 功能 | 描述 |
|--------|------|------|
| 0x11 | SLPOUT | 退出睡眠模式 |
| 0x13 | NORON | 正常显示模式开启 |
| 0x20 | INVOFF | 关闭显示反转 |
| 0x21 | INVON | 开启显示反转 |
| 0x29 | DISPON | 开启显示 |
| 0x2A | CASET | 列地址设置 (Column Address Set) |
| 0x2B | RASET | 行地址设置 (Row Address Set) |
| 0x2C | RAMWR | 写入显存 (Memory Write) |
| 0x36 | MADCTL | 存储访问控制 |
| 0x3A | COLMOD | 像素格式设置 (Interface Pixel Format) |
| 0xB2 | PORCH | Porch 设置 |
| 0xB7 | GCTRL | 栅极控制 |
| 0xBB | VCOMS | VCOM 设置 |
| 0xC0 | PWCTRL1 | 电源控制 1 |
| 0xC2 | PWCTRL2 | 电源控制 2 |
| 0xC3 | PWCTRL3 | 电源控制 3 |
| 0xC4 | PWCTRL4 | 电源控制 4 |
| 0xC6 | FRCTRL2 | 帧率控制 2 |
| 0xD0 | PWCTRL5 | 电源控制 5 |

```c
void TFTLCD_GPIO_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStructure;

    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOB, ENABLE);

    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_9 | GPIO_Pin_8 |
                                  GPIO_Pin_7 | GPIO_Pin_6 |
                                  GPIO_Pin_5 | GPIO_Pin_4;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;

                        GPIO_Pin_6 | GPIO_Pin_5 | GPIO_Pin_4);
}
```

## 九、软件模拟 SPI 写操作

```c
// 写一个字节 (MSB first)
void TFTLCD_Writ_Bus(u8 dat)
{
    u8 i;
    TFTLCD_CS_RESET;        // 片选使能
    for(i = 0; i < 8; i++)
    {
        TFTLCD_SCL_RESET;   // 时钟低
        if(dat & 0x80)      // 高位先传
            TFTLCD_SDA_SET;
        else
            TFTLCD_SDA_RESET;
        TFTLCD_SCL_SET;     // 时钟高 (上升沿采样)
        dat <<= 1;          // 左移一位
    }
    TFTLCD_CS_SET;          // 片选释放
}

// 写8位数据
void TFTLCD_WR_DATA8(u8 dat)
{
    TFTLCD_Writ_Bus(dat);
}

// 写16位数据 (RGB565)
void TFTLCD_WR_DATA(u16 dat)
{
    TFTLCD_Writ_Bus(dat >> 8);   // 先写高字节
    TFTLCD_Writ_Bus(dat);        // 再写低字节
}

// 写命令 (DC=0)
void TFTLCD_WR_REG(u8 dat)
{
    TFTLCD_DC_RESET;        // 命令模式
    TFTLCD_Writ_Bus(dat);
    TFTLCD_DC_SET;          // 切回数据模式
}
```

## 十、显示区域设置

```c
// 设置读写窗口 (地址区域)
void TFTLCD_Address_Set(u16 x1, u16 y1, u16 x2, u16 y2)
{
    // 设置列地址 (Column)
    WriteCmd(0x2A);
    WriteData(x1 >> 8);   // 起始列高字节
    WriteData(x1 & 0xFF); // 起始列低字节
    WriteData(x2 >> 8);   // 结束列高字节
    WriteData(x2 & 0xFF); // 结束列低字节

    // 设置行地址 (Row)
    WriteCmd(0x2B);
    WriteData(y1 >> 8);   // 起始行高字节
    WriteData(y1 & 0xFF); // 起始行低字节
    WriteData(y2 >> 8);   // 结束行高字节
    WriteData(y2 & 0xFF); // 结束行低字节

    // 开始写入显存
    WriteCmd(0x2C);
}
```

## 十三、使用注意事项

### 13.1 存储条件
- 温度: 23±5°C, 湿度: 45±20%RH
- 存放在防静电容器中
- 避免灰尘、活性气体和溶剂
- 不要放在有机溶剂或腐蚀性气体附近
- 避免强光照射 (会降低偏光片和滤色器性能)

### 13.2 操作注意事项
- 避免静电 (会损坏 CMOS LSI)
- 偏光片非常脆弱，请小心操作
- 不要施加外部冲击
- 不要对表面施加过大压力
- 清洁时使用棉布或软布蘸取石油醚/IPA 擦拭，不要使用干布

### 13.3 焊接要求
- 焊接温度: 320°C ~ 350°C
- 焊接时间: ≤ 10秒
- 不要在同一个位置停留过长时间

### 13.4 保修
- 保修期: 交货之日起 12 个月
- 建议 6 个月内使用完所有模块
- 存放超过 12 个月的模块，使用前建议重新检查

## 十五、与 ST7735S 的区别

| 特性 | ST7789 | ST7735S |
|------|--------|---------|
| 分辨率 | 240×240 (最大) | 128×160 (常见) |
| 适用屏幕 | 1.54寸 IPS 屏 | 1.8寸 TFT 屏 |
| 初始化序列 | 不同 (见 tftlcd.c TFT==154 分支) | 不同 (见 LCD.c) |
| RAM 大小 | 240×240×16bit | 128×160×16bit |
| 引脚兼容 | 相同 (8针SPI) | 相同 (8针SPI) |

> **注意:** 虽然两者都是 SPI 接口且引脚兼容，但初始化命令序列不同，不能混用。
